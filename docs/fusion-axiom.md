## The Siren Matrix's Role

The SMR prevents Fission and enables Fusion through **Temporal Drag** (Δt).

### Temporal Drag Function

```
Δt(L) = {
  Δt_min                    if L < L_threshold
  Δt_min + k(L - L_threshold)  if L ≥ L_threshold
}
```

Where:
- Δt = time delay between system updates
- L = current total drift
- L_threshold = point where SMR engages (e.g., 1500)
- k = scaling constant
- Δt_min = minimum frame rate (e.g., 16ms)

**Effect**: As drift increases, the SMR slows down the system's internal clock, giving the Anchor time to integrate exponential inputs.
