# Mathematical Framework: Drift Equations

## Core Equations

### 1. Exponential Drift Function

The fundamental drift at time x for a single component:

```
L(x) = e^(x·s)
```

Where:
- `L(x)` = Drift magnitude at time x
- `x` = Time elapsed in X-dimension (seconds)
- `s` = Scale factor (rate of exponential growth)
- `e` = Euler's number (~2.71828)

### 2. Total Drift with Multiple Milestones

When multiple drift sources are active:

```
L_Total(x) = Σ L_i(x) for i=1 to n
```

Where each milestone contributes:
```
L_i(x) = e^(x · s_i)  if x ≥ t_activation_i
L_i(x) = 0            if x < t_activation_i
```

### 3. Generative Power

The energy generated by the system:

```
E(x) = k · L_Total(x)
```

Where:
- `E(x)` = Generative Power at time x
- `k` = Proportionality constant
- `L_Total(x)` = Total drift magnitude

For simplicity in Beezle Bob: `k = 1`, so `E ∝ L_Total`

### 4. Siren Matrix Temporal Drag

The time delay applied by the SMR:

```
Δt(L) = Δt_min                            if L < L_threshold
Δt(L) = Δt_min + k_drag(L - L_threshold)  if L ≥ L_threshold
```

Where:
- `Δt` = Frame time (ms between updates)
- `Δt_min` = Minimum frame time (16ms for 60fps)
- `L_threshold` = Drift level triggering SMR (1500 in Beezle Bob)
- `k_drag` = Drag scaling factor (1/3 in Beezle Bob)

### 5. Fusion Ignition Condition

The system achieves fusion when:

```
FUSION = (E ≥ E_Critical) ∧ (Taxonomy = "STRANGERS WHO KNOW EACH OTHER")
```

Where:
- `E_Critical` = Minimum energy for ignition
- `∧` = Logical AND
- `Taxonomy` = Current system state classification

## Derivative Equations

### Rate of Drift Change

```
dL/dx = d/dx[e^(x·s)] = s · e^(x·s) = s · L(x)
```

**Interpretation**: The rate of drift growth is proportional to the current drift level - this is exponential growth.

### Rate of Energy Generation

```
dE/dx = k · dL_Total/dx = k · Σ[s_i · L_i(x)]
```

**Interpretation**: Energy generation rate increases as drift increases.

## Integral Equations

### Total Energy Accumulated

```
E_total = ∫[0 to x] E(t) dt = k · ∫[0 to x] L_Total(t) dt
```

For single drift source:
```
E_total = k · ∫[0 to x] e^(t·s) dt = (k/s) · [e^(x·s) - 1]
```

**Interpretation**: Total energy accumulated grows exponentially over time.

## Stability Analysis

### Critical Threshold Analysis

The system becomes unstable (fission risk) when:

```
dL/dx > (1/Δt_min) · L_threshold
```

**Interpretation**: If drift grows faster than the SMR can apply temporal drag, system risks fission.

### Stable Region

The system remains stable when:

```
L < L_threshold  ∨  (Δt(L) · dL/dx < k_stability · L)
```

Where:
- `∨` = Logical OR
- `k_stability` = Stability constant (empirically determined)

## Practical Implementation

### JavaScript Implementation

```javascript
function calculateDrift(x, milestones) {
    let totalDrift = 0;
    milestones.forEach(m => {
        if (x >= m.activationTime) {
            totalDrift += Math.exp(x * m.scale);
        }
    });
    return totalDrift;
}

function applyTemporalDrag(drift, threshold = 1500) {
    if (drift < threshold) return 16;
    return 16 + (drift - threshold) / 3;
}

}
```
